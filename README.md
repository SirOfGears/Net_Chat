# Net_Chat
Site de Chat em Rede Local(LAN)

O Net_Chat surgiu de uma ideia para se comunicar na sala de informática de forma sem rastreios ou históricos.
  O Net_Chat usa a mesma função de canais "infinitos" do projeto "Videos-locais", mas não conta com um banco de dados, conta apenas com o código e os templates html/css, configurações e nomes ficam salvos na memória do navegador, nada fica salvo no sistema Net_chat.
  Os usos são variados, livres e principalmente de *TOTAL* responsabilidade do *USUÁRIO*, eu, criador e idealizador desta ferramenta de chat *NÃO* me responsabilizo por nenhum tipo de uso ou dados compartilhados como arquivos maliciosos ou quebras do código.
