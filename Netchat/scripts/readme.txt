Como usar(se tudo estiver configurado):
1- Execute o "start_server.bat"
2- Execute o "atalho navegador.bat"

Obs.: Se já tiver um pc com o "start_server.bat" aberto, todos os outros pcs só precisam executar o "atalho navegador.bat", pois todos eles estão conectados na mesma rede, ent só precisa de um único servidor.
Obs2: Os canais são infinitos, você pode criar quantos desejar, use números, letras ou palavras para criar o canal. Por favor não use nomes impróprios. O pc que tiver o "start_server.bat" ligado, é o host, na aba do terminal que ficará aberto o host pode ver os ips de quem se conectou a cada canal, não use essa informação de forma maliciosa e nem use como chantagem saber o nome dos canais. Isso é só uma forma de manter certa ordem e saber quantos canais e usuários estão conectados. 
Obs3: Fechou o terminal host, acabou a porra toda. Sem ele, tudo morre e perde as funções.
Obs4: NADA, literalmente nada fica salvo. Ent as mensagens, os canais e os arquivos somem depois que fechar o terminal host.
